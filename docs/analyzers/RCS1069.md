## RCS1069: Remove unnecessary case label

Property | Value
--- | --- 
Id | RCS1069
Category | Redundancy
Default Severity | Hidden
Enabled by Default | yes
Supports Fade-Out | yes
Supports Fade-Out Analyzer | no

### How to Suppress

#### SuppressMessageAttribute


