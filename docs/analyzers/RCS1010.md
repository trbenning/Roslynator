# RCS1010: Use 'var' instead of explicit type \(when the type is obvious\)

Property | Value
--- | --- 
Id | RCS1010
Category | Simplification
Default Severity | Hidden
Enabled by Default | yes
Supports Fade-Out | yes
Supports Fade-Out Analyzer | no

